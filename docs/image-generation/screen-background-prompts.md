# Screen background generation prompts

These prompts create original background-only artwork for Deathrally's Phaser scenes. The uploaded Main Menu image is the visual reference: cinematic industrial night racing, near-black steel, wet reflective ground, oxide-orange practical light, restrained cyan work lights, smoke, sparks, scratched metal, and grounded photorealistic concept-art rendering.

## Shared constraints for every image

- Use case: `stylized-concept`.
- Asset type: 16:9 landscape game screen background, designed for a 1920×1080 internal canvas.
- Image 1 is a style, atmosphere, material, lighting, and quality reference only. Do not reproduce its exact layout, car, signs, title, menu, profile card, branding, or written content.
- Create an original environment with realistic scale, coherent perspective, and a polished cinematic game-background finish.
- Keep the middle and specified UI zones calm and dark enough for bright live Phaser UI to remain readable.
- No title, words, letters, numbers, logos, emblems, menu buttons, interface panels, HUD, track map, watermark, or readable signage.
- No recognizable real-world brand, trademark, copyrighted character, or copied game artwork.
- Avoid excessive orange wash, cyberpunk neon overload, fantasy technology, pristine showroom surfaces, daylight, crowds, and prominent human faces.
